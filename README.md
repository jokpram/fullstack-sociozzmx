## 📖 Tentang Proyek

**Sociozzmx** adalah aplikasi jejaring sosial *full‑stack* yang dirancang untuk menghubungkan orang‑orang dengan minat yang sama. Proyek ini menggabungkan **frontend modern** berbasis React dan TypeScript dengan **backend tangguh** menggunakan Spring Boot dan Java. Seluruh kode dikelola dalam satu repositori untuk kemudahan pengembangan dan *deployment*.

Repositori ini terdiri dari dua bagian utama:
- **`frontend/`** – Aplikasi klien yang responsif, cepat, dan interaktif.
- **`haveigone/`** – Layanan REST API yang aman, skalabel, dan terdokumentasi dengan baik.

---

## ✨ Fitur Utama

| Modul | Deskripsi |
|-------|-----------|
| 🔐 **Autentikasi & Otorisasi** | Registrasi, login, JWT, refresh token, dan role‑based access (USER, ADMIN). |
| 👤 **Profil Pengguna** | Kelola informasi profil, avatar, bio, dan pengaturan akun. |
| 📝 **Posting Konten** | Buat, edit, hapus postingan teks/gambar dengan dukungan *upload* media. |
| 💬 **Interaksi Sosial** | Like, komentar, dan bagikan postingan. |
| 🔔 **Notifikasi Real‑time** | (Opsional) WebSocket untuk notifikasi instan saat ada interaksi baru. |
| 🔍 **Pencarian & Penemuan** | Temukan pengguna lain dan konten berdasarkan kata kunci atau tagar. |
| 📊 **Dasbor Admin** | Panel khusus untuk memantau statistik dan mengelola konten/pengguna. |
| 🌐 **Dukungan Multi‑bahasa** | i18n siap pakai (Bahasa Indonesia & Inggris). |
